# 📈 A股智能分析 · 多空辩论决策系统

基于 Streamlit + Tushare + AKShare + LLM 的 A 股股票智能分析平台。

5 位不同风格的 AI 交易员（趋势 / 价值 / 短线 / 情绪 / 魔鬼代言人）实时辩论，结合三大指数、北向资金、融资融券、技术指标、财务分析与新闻事件，为你提供多维度的买卖决策参考。

## ✨ 功能特色

### 📡 数据获取（双数据源）

**Tushare（主数据源，Token 已内置）：**

- 日K线数据（前复权）+ MA5/MA20/MA60 均线系统
- 最新行情快照：最新价、涨跌幅、换手率、量比、市盈率、市净率等
- 三大指数：上证指数、深证成指、创业板指的最新行情与成交额
- 财务数据：主要财务指标（ROE/EPS/毛利率等）、利润表、资产负债表
- 资金流向：个股大单/小单买卖
- 北向资金：沪深港通近期资金流向
- 融资融券：融资余额、融资买入额、融券余额等
- 新闻公告：公司公告 + 市场快讯
- 技术指标：MACD、RSI、KDJ、布林带、波动率、最大回撤（本地计算）

**AKShare（可选实时模式，侧边栏开关控制）：**

- 通过侧边栏的「⚡ 实时数据模式」开关启用
- 开启后替换以下数据为盘中实时数据：
  - **个股行情**：最新价、涨跌幅、今开、最高、最低、成交量、换手率、市盈率等（替换首页所有行情数字）
  - **三大指数**：上证指数、深证成指、创业板指的实时点位和涨跌幅（替换指数卡片）
  - **实时K线**：当日分钟级K线汇总后合并到历史K线末尾
- 如果实时获取失败，自动回退到 Tushare 数据，不影响使用
- 交易时段使用效果最佳，盘后也可获取当日收盘数据
- 关闭时不加载 AKShare，完全不影响原有功能

### 🧠 AI 多交易员辩论系统（5+1）

| 交易员 | 风格 | 关注维度 |
|--------|------|----------|
| 📈 林顺势 | 趋势跟踪专家 | 均线系统、趋势确认、支撑阻力、大盘趋势联动 |
| 💎 陈基石 | 价值投资专家 | ROE/PE/现金流、护城河、安全边际、财务健康度 |
| ⚡ 快手哥 | 短线技术专家 | KDJ/RSI超买超卖、量价背离、龙虎榜、游资动向 |
| 🎭 赵心态 | 市场情绪专家 | 贪婪/恐惧、换手率异动、情绪极端识别、逆向择时 |
| 😈 反骨仔 | 魔鬼代言人 | 专门唱反调、找漏洞、质疑共识、搜索负面证据 |
| 👔 钱总 | 团队总经理 | 综合5方观点、特别重视反面论据、仓位/止盈止损建议 |

**AI 分析特色：**

- 每位交易员都引用三大指数、北向资金、融资融券数据作为宏观参考
- 每位交易员都会联网搜索最新新闻并纳入分析
- 内置「深度思考 + 自我反思」框架，避免确认偏误
- 魔鬼代言人专门寻找市场忽略的反面证据，确保决策全面性
- 总经理汇总时特别权衡魔鬼代言人的反向论据

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/YOUR_USERNAME/a-stock-analyzer.git
cd a-stock-analyzer
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行应用

```bash
streamlit run app.py
```

浏览器访问 `http://localhost:8501` 即可使用。

---

## 📖 使用流程

1. 在左侧边栏输入 **6位A股代码**（如 `600519` 贵州茅台）
2. （可选）开启 **「⚡ 实时数据模式」** 开关，获取盘中实时行情、指数和K线
3. 点击 **「📡 获取股票数据」** 按钮，等待数据加载完成
4. 查看三大指数概览，浏览 K线图表、技术指标、财务数据、资金流向、新闻等
5. 点击 **「🧠 启动 AI 辩论分析」** 按钮
6. 5 位交易员将依次给出深度分析，最后由总经理综合决策

## 🔧 技术栈

- **前端框架**：Streamlit + Plotly（交互式K线图表）
- **主数据源**：Tushare Pro（自定义 API 地址，Token 已内置）
- **实时数据**：AKShare（可选，侧边栏开关控制）
- **AI 模型**：gemini-3-flash（通过 OpenAI SDK 兼容接口调用）
- **AI 能力**：联网搜索 + 深度思考 + 自我反思框架
